窗口收起来了，音乐继续放。把网易云、Spotify 网页版或者喜马拉雅丢进菜单栏，不用另开 App，不用另占内存，后台一直播。支持后台播放音乐、播客等音频，直接打开音乐网站听歌，不用再下载对应的软件了。
